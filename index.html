Say-Hello-World

Esto es una aplicación móvil creada en App Inventor donde se hace "crowdsourcing" utilizando el servicio de Google Cloud y Fusion Table para hacer "data visualization" en un mapa. Los mensajes son enviados desde la aplicación, guardados en un Table Fusion y mostrados en un mapa con un pin; la app utiliza el sistema de localización GPS para obtener la geolocalización de donde el mensaje es enviado. La visualización muestra la Fecha, el mensaje, la longitud y latitud del lugar de donde se envió y estos datos son añadidos a un Fusion Table donde son almacenados.

Instrucciones

Esta aplicación es para usuarios de dispositivos móviles Android. Es un mobile apps creado con App Inventor utilizando los servicios de Google Cloud y Fusion Table para visualizar los datos en un mapa.

En esta página podrás encontrar el APK de la aplicación la cual deberás bajar e instalar en tu dispositivo. Para que la aplicación pueda ser instalada deberás otorgar el permiso acudiendo a "Settings"o "Ajustes"de tu dispositivo y seguir las siguientes instrucciones:

proceso es el siguiente:

  • Ir a "Ajustes" en el menú de aplicaciones del sistema operativo.

  • Allí escoger la opción de "Seguridad".

  • seleccionar dentro de "Seguridad" la opción de "Orígenes desconocidos" y activarla.

  • Se mostrará un mensaje de advertencia de seguridad,debes oprimir ok y aceptarla.

El habilitar esta opción te ofrecerá la posibilidad de instalar cualquier app fuera de Googe Play Store, como por ejemplo la que queremos compartir contigo a modo de prueba. Para comprobar que hayas activado la opción correctamente, descarga nuestra aplicación de prueba. Esta aplicación requiere que tu GPS este prendido para poder recibir la localización y enviarlo al mapa junto al mensajes que escribas y la fecha. Aquí te incluimos un botón de descarga del archivo APK que se usa para instalarla:   Download APK File
